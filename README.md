# bengrad
A version of micrograd https://github.com/karpathy/micrograd/blob/master/README.md (by Andrej Karpathy).
A small autograd engine with PyTorch like API. Can be used to implement deep neural networks for binary classification and multiclass
classification. Contains a few sanity checks for forward propagation and gradient checking. Has 2 examples of classification problems in the end, 1 taken from Micrograd, and the other one is inspired by it but a little different.
